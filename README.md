E-Commerce Analytics System

A SQL-based E-Commerce Analytics System developed using MySQL to manage and analyze online retail operations efficiently. This project demonstrates real-world database management concepts including customer management, product catalog management, order processing, payments, shipping, inventory tracking, product reviews, and sales analytics.

The system consists of 15 interconnected tables with a large transactional dataset and implements a wide range of MySQL concepts such as Joins, Subqueries, CTEs, Window Functions, Views, Stored Procedures, Triggers, Ranking Functions, and Business Analytics Queries. It provides valuable insights into customer behavior, revenue generation, product performance, inventory management, and overall business growth.
